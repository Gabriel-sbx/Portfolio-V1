Présentation du projet

Évolution d'un site dynamique avec CodeIgniter pour la gestion de l’Atelier Numérique

Catégorie

Plateforme de gestion , back-end, statistique, CMS, framework CodeIgniter

Technologies Utilisées

  • HTML
  • CSS
  • JavaScript
  • Bootstrap
  • Framework CodeIgniter
  • Architecture HMVC
  • PHP
  • Smarty
  • Composer (gestionnaire de dépendances)
  • MySQL
  • PhpMyAdmin
  • Git (gestion de version)
  • Visual Studio Code
  • WAMP

Présentation du projet

Ce projet de stage a été réalisé au sein de l'association Manne Emploi, dans le cadre de l’évolution de la plateforme www.atenu.fr de l’Atelier Numérique. L’objectif principal était de transformer une plateforme statique en un système de gestion de contenu (CMS) dynamique, tout en répondant aux besoins des utilisateurs : visiteurs, référents, administrateurs, et à terme les bénéficiaires

Un système modulaire a été mis en place, avec gestion des sections de page, tableau de bord statistiques, filtres avancés, gestion des bénéficiaires et modules personnalisés. Le développement s’est appuyé sur une architecture HMVC pour séparer les responsabilités et faciliter la maintenabilité.

Résultats

Ce projet m’a permis de mettre en œuvre les bonnes pratiques du développement back-end moderne, d’appliquer la méthode Merise pour l’évolution de la base de données, et d’intégrer un système de gestion de contenu modulaire. J’ai aussi pu renforcer mes compétences en sécurité (validation serveur, CSRF), gestion de projet agile (Notion, GitHub) et création d’interfaces utilisateur accessibles.
