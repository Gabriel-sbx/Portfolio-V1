Présentation du projet

Site dynamique avec symfony pour organiser des tournois Esport

Catégorie

Plateforme de gestion , back-end, système d'authentification, framework Symfony

Technologies Utilisées

  • HTML
  • CSS
  • JavaScript
  • Bootstrap
  • Framework Symfony
  • Architecture MVC
  • PHP
  • Twig
  • Composer (gestionnaire de dépendances)
  • MySQL
  • PhpMyAdmin
  • Git (gestion de version)
  • Visual Studio Code
  • WAMP

Présentation du projet

Cette application web fictive dédiée à l'organisation de tournois Esport a été développée en PHP avec Symfony et MySQL. Le projet suit une architecture MVC avec une approche code first et diagramme UML pour structurer les relations métier. L'application intègre un système d'authentification multi-rôles (administrateurs, organisateurs, joueurs) avec interface d'administration pour la gestion des utilisateurs et événements.

Résultats

Ce projet a permis de développer une application web fonctionnelle, intégrant les bonnes pratiques du développement back-end moderne. Grâce à Symfony, l’architecture est robuste, et l’approche code first a facilité l’évolution du modèle de données. Le système d’authentification et la gestion des rôles assurent une bonne séparation des accès, adaptée à un environnement compétitif comme l’Esport.
